# CX-4242-Data-and-Visual-Analytics


Course Goals:
1. Learn visual and computation techniques and tools, for typical data types
      Learn how to complement each kind of methods
      Gain a breadth of knowledge
2. Work on real datasets and problems
3. Learn practical know-how (useful for jobs, research) through significant hands-on programming assignments

Overview:

Course in data analytics and data visualization covering a broad range of topics and libraries.

Course Topics:

HW1: Collecting & visualizing data, SQLite, D3 warmup, OpenRefine
HW2: D3 Graphs and Visualization
HW3: Hadoop, Spark, Pig and Azure
HW4: Scalable PageRank via Virtual Memory (MMap), Random Forest, Scikit-Learn
